# EssentialsLib
A library that provides various features (planned) for Minecraft: Bedrock Edition.

## Features
- Hitbox detection (CollisionVolume)
  - AABB (a box shape like the one used in Minecraft)
  - Cone (conical shape)

## Usage
Download the latest version of .js, .d.ts, .js.map, and license.md from [Release](https://github.com/ma27inranma/BDS-Addon-EssentialsLib/releases)
Create a folder where it can be referenced from Script API and place them inside
Import `essentialslib.js`
